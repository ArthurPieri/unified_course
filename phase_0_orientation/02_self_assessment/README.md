## Concepts

### What this assessment is (and isn't)
It's a gap finder, not a gate. A score ≥ 70% means Phase 1 will feel like review — you can skim it for the topics you missed. A score < 70% means you need Phase 1 in full.

**Topics covered in the quiz:**
- SQL: window functions, CTEs, joins, EXPLAIN
- Python: generators, virtual envs, `pyproject.toml`, pytest basics
- Git: branch / merge / rebase / resolve-conflict
- Networking: ports, DNS, HTTP vs. HTTPS, TCP vs. UDP
- Linux/bash: permissions, piping, `jq`, exit codes

If you haven't worked in one of these areas in 12+ months, expect gaps — that's the signal this quiz exists to give you.

### Certification target — pick one, write it down
The four realistic certification paths this course supports:

| Cert | Vendor | Prereqs | Notes |
|---|---|---|---|
| **LFCA** (optional) | Linux Foundation | Phase 1 | Linux basics — cheap, vendor-neutral, good confidence builder |
| **AWS DEA-C01** | AWS | Phases 0–5 + vendor AWS branch | Most market demand in 2026 |
| **Azure DP-700** | Microsoft | Phases 0–5 + vendor Azure branch | Fabric-centric; grows in enterprises on MS stack |
| **Snowflake tri-cert** | Snowflake | Phases 0–5 + vendor Snowflake branch | SOL-C01 → COF-C02 → DEA-C02 progression |

Official cert pages: [../../references/docs.md § Vendor certification pages](../../references/docs.md).

### Write your learning goal
Create `goal.md` in this directory. One paragraph, four sentences max:
1. The cert you're targeting
2. Why (job change, promotion, credibility, curiosity)
3. Your rough timeline
4. Your hardware profile (full, light, or cloud fallback)

This becomes your anchor during Phase 3+ when the material gets heavy.
