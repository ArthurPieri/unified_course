# Phase 0 — Orientation and Mental Models (5–8h)

Before learning any tool, build the map. This phase gives you the vocabulary, the historical arc, and the readiness check so Phase 1+ has somewhere to stick.

| # | Name | Hours | Link |
|---|---|---|---|

| 04 | Cloud fallback — Codespaces / Gitpod | 1h | [04_cloud_fallback/](04_cloud_fallback/) |

## Exit criteria
- Self-assessment ≥ 70% (see `02_self_assessment/quiz.md`)
- Hardware verified OR cloud fallback configured
- Target certification chosen and written down in `02_self_assessment/goal.md` (create your own)
- Exit quiz passed — [quiz.md](quiz.md)
